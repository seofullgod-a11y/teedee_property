## ฟีเจอร์

### หน้าบ้าน (ผู้เข้าชม)
- **เสิร์ชบาร์หลัก** — แท็บ เช่า/ซื้อ/ทั้งหมด + คำค้น + ประเภททรัพย์ + ช่วงงบประมาณ + ชิปคำค้นยอดนิยม
- **หน้าค้นหา** — กรองตามประเภท ราคา ห้องนอน คำค้น / เรียงตามราคา-มาใหม่-ยอดนิยม / แบ่งหน้าอัตโนมัติ
- **หน้ารายละเอียด** — แกลเลอรีรูป, สเปคครบ (ห้องนอน/น้ำ/ตร.ม./ตร.ว./ชั้น/สัตว์เลี้ยง), จุดเด่น, สถานที่ใกล้เคียง (BTS/MRT/ห้าง), ประกาศใกล้เคียง
- **ฟอร์มติดต่อ** — ผู้สนใจฝากชื่อ เบอร์ LINE ID ข้อความ ส่งเข้าระบบหลังบ้านทันที
- **รายการโปรด ❤️** — กดหัวใจเก็บประกาศ ดูได้ที่ `/saved` (เก็บบนเครื่อง ไม่ต้องสมัครสมาชิก)
- **ดูล่าสุด** — หน้าแรกจำประกาศที่เพิ่งเปิดดูให้อัตโนมัติ
- **เครื่องคำนวณผ่อนบ้าน** — สไลเดอร์ เงินดาวน์/ดอกเบี้ย/ระยะกู้ บนประกาศขายทุกอัน
- **ติดต่อตรง** — ถ้าประกาศมีเบอร์/LINE ID จะมีปุ่มโทรและแอด LINE ทันที
- **แชร์** — คัดลอกลิงก์ / แชร์เข้า LINE / native share บนมือถือ
- **SEO พร้อม** — OG tags รายประกาศ (แชร์ลิงก์แล้วขึ้นรูป+ราคา), sitemap.xml, robots.txt, canonical ทุกหน้า
- **🚀 หน้าทำเลอัตโนมัติ (SEO)** — สร้างหน้า Landing ตามทำเลให้เองจากประกาศในระบบ เช่น `/area/กรุงเทพมหานคร`, `/area/กรุงเทพมหานคร/rent/condo` ("คอนโดเช่า กรุงเทพฯ") พร้อม H1/คำโปรยที่ไม่ซ้ำ (สร้างจากข้อมูลจริง: จำนวน ราคาเริ่มต้น หมวดที่มี), ชิปกรองในทำเล, ลิงก์เชื่อมทำเลอื่น (internal linking) และหน้า `/areas` รวมทุกทำเล — เพจว่างจะตั้ง `noindex` อัตโนมัติกันเนื้อหาบาง
- **📊 Structured Data (JSON-LD)** — หน้าประกาศฝัง `Product`+`Offer`+`BreadcrumbList` (ขึ้นราคา/รูปใน Google), หน้าแรกฝัง `WebSite`+`SearchAction` (ช่องค้นหาใน Google) และ `RealEstateAgent`, หน้าทำเลฝัง `CollectionPage`+`ItemList` — sitemap รวมหน้าทำเลทุกชุดให้เอง
- **🗺️ Google Maps** — แผนที่ฝังบนหน้าประกาศ + ปุ่ม "เปิดใน Google Maps" และ "นำทาง" (หลังบ้านวางลิงก์ Maps แล้วระบบแกะพิกัดให้อัตโนมัติ ไม่มีพิกัดก็ใช้ชื่อทำเลแทน)
- **🧭 สำรวจย่านนี้ (ใต้แมพ)** — จัดหมวดสถานที่ใกล้เคียงอัตโนมัติ (เดินทาง/ช้อปปิ้ง/สุขภาพ/การศึกษา/ไลฟ์สไตล์) + **AI วิเคราะห์ทำเล** (แคชต่อจังหวัด สร้างครั้งเดียวใช้ซ้ำ ประหยัด API, ไม่มี key ก็มีเวอร์ชัน fallback) + ลิงก์ไปหน้าทำเล SEO
- **🚗 เวลา/ระยะเดินทาง** — ประเมินระยะเส้นตรงถึงฮับสำคัญ (สนามบิน ใจกลางเมือง ฯลฯ) แบบฟรีไม่ต้องมี API key + ช่องพิมพ์จุดหมายเองเพื่อเปิดเส้นทางจริงใน Google Maps
- **📅 นัดชมสถานที่จริง** — เลือกวัน-ช่วงเวลา + ชื่อ/เบอร์ ส่งเข้ากล่องข้อความหลังบ้าน + แจ้ง Telegram ทันที
- **🏘️ ทรัพย์อื่นในทำเลเดียวกัน** — แถวการ์ดประกาศในจังหวัดเดียวกันโดยเฉพาะ + ลิงก์ดูทั้งหมดในทำเล
- **🌙 Dark Mode (ทุกหน้า)** — สลับโหมดมืด/สว่างได้ทั้งเว็บรวมหลังบ้าน จำค่าไว้ใน localStorage + สคริปต์กันจอกระพริบ (anti-flash) ก่อนหน้าโหลด ปรับผ่าน CSS variables ล้วน
- **🌐 สองภาษา ไทย/อังกฤษ** — ปุ่มสลับภาษาบนแถบเมนู แปล UI ทั้งหมด (เมนู ปุ่ม หัวข้อ ฟอร์ม ป้ายกำกับ) ผ่าน dictionary + auto-localizer (MutationObserver) ส่วนเนื้อหาที่เจ้าของกรอก (ชื่อประกาศ/รายละเอียด/ทำเล) คงภาษาเดิมไว้ — หน้า admin เป็นภาษาไทย (เครื่องมือเจ้าของ) แต่รองรับ Dark Mode
- **✨ AI Search** — พิมพ์ภาษาคนบนหน้าแรก เช่น "คอนโดเลี้ยงแมวได้ ใกล้ BTS ไม่เกิน 2 หมื่น" AI แปลงเป็นฟิลเตอร์ให้เอง (ต้องมี ANTHROPIC_API_KEY, ไม่มีก็ fallback เป็นค้นหาปกติ)
- **💬 AI Chat ประจำประกาศ** — ปุ่มแชทลอยขวาล่างบนหน้าประกาศ AI ตอบคำถามจากข้อมูลประกาศจริง (rate limit 30 ข้อความ/ชม./IP) พร้อมเก็บ lead: ถามชื่อ-เบอร์ในแชทแล้วส่งเข้ากล่องข้อความ + แจ้ง Telegram
- **แกลเลอรีใหม่ + Lightbox** — รูป grid สไตล์ Airbnb คลิกดูเต็มจอ เลื่อนด้วยลูกศร/ปัดนิ้ว
- **Favicon = โลโก้** — อัปโหลดโลโก้ในหลังบ้านแล้ว favicon เปลี่ยนตามอัตโนมัติ
- **เมนูล่างแบบแอป (มือถือ)** — หน้าแรก/เช่า/ซื้อ/ที่ดิน/โปรด ลอยด้านล่าง สไตล์ iOS
- **ทำเลยอดนิยม** — หน้าแรกดึงจังหวัดที่มีประกาศมากสุดจากฐานข้อมูลอัตโนมัติ
- นับยอดวิวต่อประกาศ (แสดงบนหน้าประกาศพร้อมวันที่ลง), ป้าย "ประกาศแนะนำ", responsive รองรับมือถือ

### หลังบ้าน (`/admin`)
- ล็อกอินด้วยรหัสผ่าน (ตั้งผ่าน Environment Variable)
- **แดชบอร์ด** — จำนวนประกาศ ยอดวิวรวม ข้อความติดต่อ/ยังไม่อ่าน
- **จัดการประกาศ** — เพิ่ม/แก้ไข/ลบ, สถานะ เผยแพร่/ร่าง/ปิด, ปักดาวประกาศแนะนำ, รูปหลายรูป, สถานที่ใกล้เคียง
- **🤖 AI ช่วยคิดคอนเทนต์** (Claude API)
  - ✨ คิดชื่อประกาศ 5 แบบ (เลือกใช้ได้ทันที)
  - 📝 เขียนคำอธิบายประกาศ
  - 📌 สรุปจุดเด่นเป็นข้อ ๆ
  - 📣 เขียนแคปชันโพสต์ Facebook/LINE (กดคัดลอกได้เลย)
- **กล่องข้อความ** — อ่านข้อความจากผู้สนใจ พร้อมลิงก์ไปประกาศที่เกี่ยวข้อง
- **ตั้งค่าเว็บไซต์** — เปลี่ยนโลโก้ (URL รูป), ชื่อเว็บ 2 ส่วน (ปกติ+สีทอง), ข้อความรอง พร้อมพรีวิวสด มีผลทั้งเว็บทันที
- **จัดการเร็ว** — ปุ่มเปิด/ปิดประกาศ 1 คลิก และปุ่ม "สำเนา" คัดลอกประกาศเดิมเป็นฉบับร่าง (สะดวกเวลาลงห้องหลายยูนิตในตึกเดียว)
- **🔔 แจ้งเตือน Telegram** — มีคนทักปุ๊บ บอทส่งข้อความหาคุณทันที (ชื่อ เบอร์ LINE ข้อความ และประกาศที่สนใจ)

## Deploy บน Railway (ผ่าน GitHub)

1. **อัปโหลดขึ้น GitHub** — สร้าง repo ใหม่ แล้วอัปโหลดไฟล์ทั้งหมดผ่าน GitHub web editor (โครงสร้างตามด้านบน ให้ `server.js` อยู่ root ของ repo)

2. **สร้างโปรเจคใน Railway**
   - New Project → Deploy from GitHub repo → เลือก repo นี้

3. **เพิ่มฐานข้อมูล**
   - ในโปรเจคเดียวกัน กด **+ New → Database → PostgreSQL**
   - Railway จะสร้างตัวแปร `DATABASE_URL` ให้อัตโนมัติ — ไปที่ service ของแอป → Variables → **Add Reference** → เลือก `DATABASE_URL` จาก Postgres

4. **ตั้งค่า Variables ของแอป**

   | ตัวแปร | ค่า | จำเป็น |
   |---|---|---|
   | `DATABASE_URL` | reference จาก PostgreSQL | ✅ |
   | `ADMIN_PASSWORD` | รหัสเข้าหลังบ้าน (ตั้งเอง อย่าใช้ค่า default) | ✅ |
   | `ANTHROPIC_API_KEY` | คีย์จาก console.anthropic.com | สำหรับฟีเจอร์ AI |
   | `ANTHROPIC_MODEL` | (ไม่ใส่ก็ได้) default `claude-sonnet-4-6` | – |
   | `TELEGRAM_BOT_TOKEN` | โทเคนบอทจาก @BotFather | แจ้งเตือนเมื่อมีคนติดต่อ |
   | `TELEGRAM_CHAT_ID` | chat id ที่ให้บอทส่งข้อความหา | แจ้งเตือนเมื่อมีคนติดต่อ |
   | `BASE_URL` | เช่น `https://teedee.up.railway.app` (สำหรับ sitemap/OG) | แนะนำ |

5. **Deploy** — Railway จะรัน `npm start` อัตโนมัติ ตารางและข้อมูลตัวอย่าง 13 รายการจะถูกสร้างให้เองตอนบูตครั้งแรก

6. เข้าใช้งาน
   - หน้าเว็บ: `https://<domain>.up.railway.app`
   - หลังบ้าน: `https://<domain>.up.railway.app/admin`

> ⚠️ ถ้ายังไม่ตั้ง `ADMIN_PASSWORD` ระบบจะใช้ค่าชั่วคราว `teedee1234` — **ต้องเปลี่ยนก่อนเปิดใช้จริงเสมอ**

## การใส่รูปประกาศ

ระบบใช้ลิงก์รูป (URL) — แนะนำอัปโหลดรูปขึ้นบริการฟรีอย่าง [postimages.org](https://postimages.org) หรือ imgbb.com แล้วคัดลอก **Direct link** มาวางในหลังบ้าน (เพิ่มได้หลายรูปต่อประกาศ)

## รันทดสอบในเครื่อง

```bash
npm install
# ต้องมี PostgreSQL: postgresql://teedee:teedee@localhost:5432/teedee (หรือ set DATABASE_URL)
npm start
# เปิด http://localhost:3000
```
